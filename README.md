# 知っている範囲内でしか訳さないアプリ（プロトタイプ）

ローカルの単語帳（IndexedDB）に登録した語彙だけを使い、**未登録箇所を明示（ハイライト）**しながら「いまの自分の学習範囲」で訳すためのプロトタイプです。

## ねらい（コンセプト）

- **単語帳**: 人が登録するのは最小限（見出し語・品詞・意味・活用タイプ）
- **文法エンジン（試作）**: 文のときは単語の置換だけではなく、活用タイプとルールで文を生成
  - 例: `Αγοράζω μήλο.` → `私は りんごを 買う。`

## できること

- **単語モード**: 単語帳に登録済みの語彙だけで置換し、未登録箇所をハイライト
  - 動詞/名詞は、見出し語＋活用タイプから生成できる範囲の活用形でも照合します（例: `αγοράζεις` など）
- 形容詞/疑問詞（最小対応）は、見出し語から生成できる範囲の活用形でも照合します
- **文モード（試作）**: ギリシャ語→日本語で、最小ルールにより省略主語/助詞を補って文を生成
  - 動詞（`-ω / -εις / -ει / -ουμε / -ετε / -ουν(ε)`）→ `私は / あなたは / 彼/彼女は / 私たちは / あなたたちは / 彼ら/彼女らは`
  - 名詞は簡易に格を推定し、`を / が` を補う（未確定な場合は位置ヒューリスティック）
- **単語帳**:
  - メモ、例文、関連語、タグ（複数可）
    - 例文は **「原文＋訳」** のペアで保存できます（詳細画面では原文/訳を表示）
    - 文モードでは、入力文が例文（原文）と **完全一致**し、かつ訳が登録されている場合は、その訳を最優先で返します（最小実装）
  - 活用（マトリックス）表示＋編集（上書きした形は単語モード/文モードの照合にも反映）
  - 一覧フィルター（ウインドウ式）: **品詞 / タグ / アルファベット（ギリシャ文字Α〜Ω）**
  - 変化する語尾を赤字で強調表示（動詞/名詞/形容詞）
  - 単語削除は確認モーダルを表示（画面中央）
- 形容詞/疑問詞（試作）:
  - 男性形（例: `μεγάλος`）を登録すると、女/中（＋複数・格）をマトリックス表示（最小対応）
  - 疑問詞（例: `ποιος`, `πόσος`）は形容詞のように活用表を表示・照合します
  - `πολύς` のような不規則も最小対応しています
- **設定**:
  - 設定トップに **表示言語 / タグ設定 / 一括登録 / バックアップ** の4ボタン
  - タグ設定: タグ追加、プルダウンで選択して削除（単語側のタグは残ります）
  - バックアップ（JSON）: エクスポート / インポート（上書き）
  - 一括登録（Excel）: テンプレをダウンロードして `.xlsx` をアップロード（後述）
- **重複チェック**:
  - 既に登録済みのタグ/単語を追加しようとすると、モーダルでメッセージを表示します

## 技術スタック

- React + TypeScript + Vite
- IndexedDB（Dexie）
- （任意）Supabase（Auth + DB）: クラウド同期（PC/スマホで同じ単語帳を使う）

## コード構成（ディレクトリ）

UIの大きな機能は `src/features/` に寄せています。

- `src/features/wordbook/`: 単語帳（一覧/詳細/編集/活用表）
- `src/features/translate/`: 翻訳（単語モード/文モードのUI）
- `src/features/translate/services/`: 翻訳サービス（DBアクセスやI/Oを担当）
- `src/grammar/`: 活用生成・文モードの純ロジック（できるだけI/Oから分離）
- `src/lib/`: 小さなユーティリティ（例: `tokenize`）

## コード構成（文法/活用まわり）

このプロトタイプでは、活用生成やアクセント処理を `src/grammar/` に寄せています（UIは `src/features/` 側）。

- **`src/grammar/accent.ts`**
  - トノスの除去/付与、アクセント位置推定
  - **母音ユニット**（例: `ου/ευ/αι/ει/οι/αυ/ηυ` など）を 1 つとして扱うAPIも持ちます
    - 例: `addTonosOnNthFromEndVowelUnit(...)`
    - 例: `applyAccentFromByVowelUnit(...)`
- **`src/grammar/verb.ts`**
  - 動詞の最小活用（現在/過去/未来/να/命令、アオリスト）を生成
  - UIラベルは持たず、`person: '1sg' | ...` のような **純データ**を返します
- **`src/grammar/noun.ts`**
  - 名詞の自動語形（`nounAutoForms`）とマトリックス用データ（`nounMatrix`）を生成
  - `endingsByCell`（語尾表）や `resolveNounTypeForMatrix`（名詞タイプ決定）もここに集約
  - `nounMatrix` は **冠詞なし・UIラベルなし**の語形データを返し、冠詞表示（`ο/η/το` 等）や「単数/複数」「～は/～の/～を」は UI 側で組み立てます
- **`src/grammar/adjective.ts`**
  - 形容詞/疑問詞の最小活用（マトリックス/照合用の語形集合）を生成
- **`src/grammar/engine.ts`**
  - 文モードのコア（純ロジック）: `translateSentenceForeignToJaCore(...)`
- **`src/features/translate/services/sentenceTranslate.ts`**
  - 文モードのI/O（DB読み/例文完全一致の優先など）: `translateSentenceForeignToJa(...)`
- **`src/features/translate/services/knownOnlyTranslate.ts`**
  - 単語モード翻訳: `translateKnownOnly(...)`

## 使い方（最短）

1. `登録` で単語を追加
   - 品詞: **名詞 / 動詞 / 形容詞 / 副詞 / 人称代名詞 / 疑問詞 / 前置詞 / 接続詞 / その他**
   - **動詞**は活用タイプを選択（例：`Α : -νω/-ζω` など）
     - 活用タイプが **その他** の場合は、活用は編集画面で手入力してください（活用表も手入力で表示します）
   - **名詞**は性（男/女/中）を選択（活用タイプは見出し語の語尾から自動推定）
   - （任意）タグ/例文/関連語/メモを追加
2. `翻訳` で
   - 単語単位なら **モード=単語**
   - 文なら（外国語→日本語で） **モード=文（試作）**
3. `単語帳` で詳細確認・編集（関連語はタップでジャンプ）
4. `設定` で
   - `表示言語`（※現状は日本語のみ対応）
   - `タグ設定`（タグの追加/削除）
   - `一括登録`（Excelのアップロードでまとめて登録）
   - `バックアップ`（JSONのエクスポート/インポート）

## データ保存とバックアップについて

- **保存先**: ブラウザの IndexedDB（DB名: `knownOnlyTranslate`）
- **バックアップ（JSON）**: `設定 > バックアップ` からエクスポート/インポートできます
  - エクスポートは「単語帳（entries）」と「設定（settings）」を1つのJSONに保存します
  - インポートは **現在データを削除して復元（上書き）** します
  - 形式は概ね以下です（実体は `settings` / `entries` を含むオブジェクト）
    - `kind: "known-only-translate-backup"`
    - `version: 1`

## 一括登録（Excel）

`設定 > 一括登録` から、テンプレExcelを使ってまとめて登録できます。

- **テンプレ**: 画面上の「テンプレートをダウンロード」から取得できます（`template.xlsx`）
- **必須シート**: `word`, `noun`, `adjective`, `verb`
- **必須列（wordシート）**: `foreignLemma`, `pos`, `meaningJa`
- **品詞ごとの必須行**
  - `pos=noun` の行は、`noun` シートに同じ `foreignLemma` の行が無いとエラー
  - `pos=verb` の行は、`verb` シートに同じ `foreignLemma` の行が無いとエラー
  - `pos=adjective` / `pronoun_interrogative` の行は、`adjective` シートに同じ `foreignLemma` の行が無いとエラー
- **エラー表示**: ダイアログに `word` の「何行目の何が不足か」を一覧表示します
- **重複lemma**: 既存の `foreignLemma` と重複した場合、ダイアログで **上書き / スキップ / キャンセル** を選べます
- **活用の自動生成（空欄補完）**
  - `noun` / `adjective` / `verb` シートの活用欄が空欄の場合、見出し語とタイプから生成できる範囲で自動生成し、照合用の `foreignForms` に反映します
  - 動詞の `inflectionType=none（その他）` は自動生成しません（手入力前提）

### Supabase側の準備（メール+パスワード + 1ユーザー=1単語帳）

- Authenticationで **Email（メール/パスワード）** を有効化
- DBに `user_state` テーブル（バックアップJSONを丸ごと保存）を作成し、RLSで本人だけアクセスできるようにします

```sql
create table if not exists public.user_state (
  user_id uuid primary key references auth.users (id) on delete cascade,
  data jsonb not null,
  updated_at timestamptz not null default now()
);

alter table public.user_state enable row level security;

create policy "user_state_select_own"
on public.user_state for select
using (user_id = auth.uid());

create policy "user_state_insert_own"
on public.user_state for insert
with check (user_id = auth.uid());

create policy "user_state_update_own"
on public.user_state for update
using (user_id = auth.uid())
with check (user_id = auth.uid());

create policy "user_state_delete_own"
on public.user_state for delete
using (user_id = auth.uid());
```

## 動詞活用タイプ（試作）

現状、動詞は以下の活用タイプを選べます（増加中）。

- **`その他`**（内部: `none`）: 活用表は手入力（上書き）で運用
- **`Α : -νω/-ζω`**（内部: `verb_pres_act_-ω`）
- **`Α : -γω/-χω/-χνω`**（内部: `verb_pres_act_-γω_-χω_-χνω`）
- **`Α : -πω/-φω/-βω/-εύω`**（内部: `verb_pres_act_-πω_-φω_-βω_-εύω`）
- **`ΑΒ : -ω（αόρ. 不規則）`**（内部: `verb_pres_act_AB`）
- **`Β1 : -άω（αόρ. -ησα）`**（内部: `verb_pres_act_B1_-άω_-ησα`）
- **`Β1 : -άω（αόρ. -εσα）`**（内部: `verb_pres_act_B1_-άω_-εσα`）
- **`Β1 : -άω（αόρ. -ασα）`**（内部: `verb_pres_act_B1_-άω_-ασα`）
- **`Β2 : -ώ（αόρ. -ησα）`**（内部: `verb_pres_act_B2_-ώ_-ησα`）
- **`Β2 : -ώ（αόρ. -ασα）`**（内部: `verb_pres_act_B2_-ώ_-ασα`）
- **`Β2 : -ώ（αόρ. -εσα）`**（内部: `verb_pres_act_B2_-ώ_-εσα`）
- **`Γ1 : -ομαι`**（内部: `verb_pres_mid_Γ1_-ομαι`）
- **`Γ2 : -άμαι`**（内部: `verb_pres_mid_Γ2_-άμαι`）

### Γ（-ομαι / -άμαι）の最小ルール（表示・照合向け）

例:

- `εργάζομαι → εργάστηκα`（Γ1）
- `κοιμάμαι → κοιμήθηκα`（Γ2）

対応範囲（最小）:

- **現在（継続・繰返し）**
  - Γ1: `-ομαι/-εσαι/-εται/-όμαστε/-εστε/-ονται`
  - Γ2: `-άμαι/-άσαι/-άται/-όμαστε/-άστε/-όνται`
- **過去（継続・繰返し）**（未完了過去・最小）: `-όμουν/-όσουν/-όταν/-όμασταν/-όσασταν/-όνταν`
- **未来（継続・繰返し）**: `θα + 現在`（表示用）
- **アオリスト過去**（最小）
  - Γ1: `-στηκα` 系（例: `εργάζομαι → εργάστηκα`。`...ζ` は `ζ→σ` を入れて生成）
  - Γ2: `-ήθηκα` 系（例: `κοιμάμαι → κοιμήθηκα`）
- **アオリスト未来**（最小）
  - Γ1: `θα εργαστώ` 系
  - Γ2: `θα κοιμηθώ` 系
- **命令（2人称のみ・最小）**
  - 継続・繰返し: 2複のみ（Γ1: `...εστε` / Γ2: `...άστε`）
  - アオリスト: 2単/2複（例: `εργάσου/εργαστείτε`, `κοιμήσου/κοιμηθείτε`）

### 活用マトリックスの列

動詞のマトリックスは、**「να」の右に命令形（命令）**を表示します。

- **命令列**は **2人称（2単/2複）のみ**が対象で、それ以外（1人称/3人称）は空欄になります。

### Β1（-άω）の最小ルール（表示・照合向け）

例:

- `απαντάω / βοηθάω / ζητάω`（-ησα 型）
- `γελάω / πεινάω / χαλάω`（-ασα 型）

対応範囲（最小）:

- **現在（継続・繰返し）**: `-άω/-άς/-άει/-άμε/-άτε/-άνε`（`απαντά` も 3単として許容）
- **過去（継続・繰返し）**: `-ούσα` 系（例: `απαντούσα`）
- **アオリスト過去**:
  - `-ησα` 型: `απαντάω → απάντησα`
  - `-εσα` 型: `πονάω → πόνεσα` / `φοράω → φόρεσα`
  - `-ασα` 型: `γελάω → γέλασα`
- **命令（2人称のみ）**
  - Imperfective: 2単 `-α`（例: `απάντα`）、2複 `-άτε`（例: `απαντάτε`）
  - Perfective: 2単 `-ησε / -ασε`（例: `απάντησε / γέλασε`）、2複 `-ήστε / -άστε`（例: `απαντήστε / γελάστε`）

### Β2（-ώ）の最小ルール（表示・照合向け）

例:

- `δημιουργώ → δημιούργησα`（-ησα 型）
- `εξηγώ → εξήγησα`（-ησα 型）

対応範囲（最小）:

- **現在（継続・繰返し）**: `-ώ/-είς/-εί/-ούμε/-είτε/-ούν`
- **過去（継続・繰返し）**: `-ούσα` 系（例: `δημιουργούσα`）
- **アオリスト過去**:
  - `-ησα` 型: 語幹 + `ησα`（例: `δημιουργώ → δημιούργησα`）
  - `-ασα` 型: 語幹 + `ασα`（最小ルール）
  - `-εσα` 型: 語幹 + `εσα`（最小ルール）
- **命令（2人称のみ）**
  - Imperfective: 2単 `-εί`、2複 `-είτε`
  - Perfective: 2単 `-σε`、2複 `-στε`（アオリスト語幹から生成）

### 命令形のトノス（アクセント）ルール（試作）

命令形のトノス位置は「母音ユニット数」で決めます。

- **母音ユニット**: `εύ/ου/αι/ει/οι/αυ/ηυ` を **1つ**として数えます（例: `δουλεύω` は `ου + εύ + ω` で 3）。
- **2単（命令）**:
  - 母音ユニット数が **2以下** → 後ろから **2番目**
  - それ以外 → 後ろから **3番目**
- **2複（命令）**: 後ろから **2番目**

### アオリスト（ψ/ξ 系）の最小ルール

活用タイプによっては、語幹変化を最小ルールで扱います。

- `-γω/-χω/-χνω` → **ξ 系**（例: `τρέχω → έτρεξα`）
- `-πω/-φω/-βω/-εύω` → **ψ 系**（例: `γράφω → έγραψα`, `χορεύω → χόρεψα`）

### 語尾ハイライト（赤/青）

単語帳の活用マトリックスでは、**人称語尾を赤**、（アオリスト等の）**目印を青**で強調します。

- **赤（語尾）**: 人称語尾（例: `αγόρασα` の `α`、`θα αγοράσω` の `ω`）
- **青（目印）**:
  - アオリストの `σ/ψ/ξ`（例: `αγόρα` + `σ` + `α`）
  - Β1（-ησα）の `ησ`（例: `απάντ` + `ησ` + `α`）
  - Γ（中動態・最小）の `στ/ηθ`（例: `εργα` + `στ` + `ηκα`、`κοιμ` + `ηθ` + `ηκα`）
  - Β1/Β2 未完了過去の `ούσ`（例: `απαντ` + `ούσ` + `α` / `δημιουργ` + `ούσ` + `α`）
  - ΑΒ 未完了過去/命令（2単）の `γ`（例: `έκαι` + `γ` + `α`、`καί` + `γ` + `ε`）

※ どれも厳密なギリシャ語活用の完全実装ではなく、単語帳の表示・照合を目的にしたヒューリスティックです。例外は「活用の上書き」で対応できます。
